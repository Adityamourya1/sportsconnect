# Frontend - SportsNet

React frontend for the sports social media platform.

## Setup
```bash
cd frontend
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Features
- User authentication
- Personalized feed
- Post creation with AI assistance
- Explore trending content
- League management
- User profiles
- Real-time interactions

## Environment Variables
Create `.env.local`:
```
VITE_API_URL=http://localhost:8000/api
```
